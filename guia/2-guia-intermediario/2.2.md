### manipulando Dados Compostos

~~~ alg
Algoritmo "criando vetor"

Var
    trem : vetor [1..10] de logico
    busca : inteiro

Inicio
  busca <- 1
  Enquanto (busca <= 10) Faca
    trem[busca] <- verdadeiro
    busca <- busca + 1
  FimEnquanto

FimAlgoritmo
~~~
